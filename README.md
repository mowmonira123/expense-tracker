# Expense Tracker

A simple command-line tool to track personal expenses.

## Features
- Add expenses
- View all expenses
- View summary
- View graph (matplotlib)
- Show remaining budget
- Update limit anytime
- Monthly spending limit
- Remove an expense by selecting its number
- Reset all expenses

## Installation
pip install -e 

pip install matplotlib

## Usage
python -m expense-tracker

## How to open and run
Download and unzip the file -> open the CMD file in the first folder -> follow the installation commands one by one -> follow the usage command
